# Temperature Graph

Проект для построения графика температуры с использованием аргументов командной строки.

## Использование argparse

Скрипт использует модуль `argparse` для получения параметров через командную строку. Все аргументы являются позиционными и обязательными.

### Аргументы командной строки:

1. **city_name** - название города на английском языке  
   Пример: `Minsk`, `London`, `Paris`

2. **country_code** - двухбуквенный код страны   
   Пример: `BY` (Беларусь), `US` (США), `DE` (Германия)

3. **start_date** - дата начала периода в формате YYYY-MM-DD  
   Пример: `2025-01-01`

4. **end_date** - дата окончания периода в формате YYYY-MM-DD  
   Пример: `2025-01-07`

### Пример использования:

```bash
# Получить температуру для Минска за период с 1 по 6 октября 2025 года
python main.py Minsk BY 2025-10-01 2025-10-06
